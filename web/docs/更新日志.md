[1.1.0] - 2026-03-20
- “自动布局”算法更新：引入 ASAP (As Soon As Possible, 尽可能早 / 起点对齐) 策略；支持起点对齐（保持同源分支起点整齐）或终点对齐（强推末端节点对齐）。算法会根据策略智能推导节点深度。
- “自动布局”多端口连线的上下乱序问题：当一个节点上有多个输入和输出端口时，按照端口从上到下的顺序依次上下排列节点

[1.0.9] - 2026-02-03
- "快速连线"，增加了对“端口为多个类型”节点的匹配连接。

[1.0.8] - 2026-02-02
- 调整“快速连线”评分权重策略，从“惩罚机制”修改为“奖励机制”。
- 增加“快速连线” Ctrl+Shift+Alt+F 清理单个节点连线时，暴力断开该节点的所有输入和输出连线。

[1.0.7] - 2026-01-31

- "对齐面板"的水平分布间距、垂直分布间距，输入值增加了内存变量，能够继续使用上一次输入的数值，且在浏览器刷新后和ComfyUI重启后，会恢复默认值（在设置中设定的值）。

[1.0.4] - 2026-01-28
